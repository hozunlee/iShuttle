# [FE_006] 업로드 후 스냅샷 & 팀 지정

## 목표

영상 업로드 후 분석 시작 전, 스냅샷 1장을 보여주고 사용자가 "우리 팀이 어느 쪽인지" 지정할 수 있도록 한다.

## 배경

카메라 촬영 방향이 항상 일정하지 않음. "우리 팀 = 화면 하단" 가정이 틀릴 수 있어
랠리 결과(득점/실점)가 반전될 수 있다.

## 완료 기준 (Done Criteria)

- [ ] Setup 화면: 업로드 후 영상 5초 지점 스냅샷 표시
- [ ] "우리 팀이 어느 쪽?" → 상단 / 하단 버튼 선택
- [ ] 선택값이 `game_config.our_side: "top" | "bottom"` 으로 백엔드 전달
- [ ] `rally.py` assign_rally_result()가 our_side 값 반영

## 구현 방향

**FE (Setup.tsx)**
```tsx
// 업로드 완료 후 스냅샷 단계 추가
// video 태그로 5초 seeked 후 canvas.drawImage → 미리보기
// "상단이 우리 팀" / "하단이 우리 팀" 버튼
```

**BE (analyze.py → rally.py)**
```python
# game_config에 our_side 추가
our_side = game_config.get("our_side", "bottom")  # 기본값: 하단
# assign_rally_result에서 our_side 기준으로 득점/실점 판별 반전
```
