# [BE_006] 짧은 클립 영상 지원

> 상태: DONE
> 담당: BE
> 생성: 2026-03-25

## 목표

25점 전체 게임이 아닌 짧은 클립(레슨 구간, 연습 영상 등)도 정상 분석되도록 한다.

## 배경

사용자가 전체 게임 영상(20~30분)을 올리기 어렵고, 실제로는
- 좋은 랠리 구간만 따로 촬영한 클립
- 레슨 영상의 특정 섹션
- 10~60초 단위 짧은 영상

을 올리게 됨.

## 현재 문제

1. mock_rallies()가 영상 전체 길이 기반으로 랠리 개수 계산 → 30초 영상이면 랠리 1~2개만 생성
2. Setup 화면의 안내 문구가 "게임 전체 영상" 전제로 작성됨
3. 점수 집계(us/them score)가 누적 방식 → 짧은 클립에서는 의미 없음

## 완료 기준 (Done Criteria)

- [ ] 30초 이하 클립도 최소 1개 랠리 감지 (mock 포함)
- [ ] Setup 화면 안내 문구 → "게임 전체 또는 구간 클립 가능"으로 수정
- [ ] 짧은 영상에서 score_at_end 표시 방식 검토 (누적 점수 대신 "해당 랠리만" 표시 옵션)

## 구현 방향

**BE (rally.py)**
```python
# _mock_rallies: 최소 1개 보장
# 30초 미만 영상 → 영상 전체를 1개 랠리로 처리
```
